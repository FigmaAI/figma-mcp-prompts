# Prompt

선택한 노드의 모든 텍스트를 스캔하고, 첨부된 이미지에서 사이트 맵 텍스트를 추출한 후, 모든 텍스트를 한 번에 업데이트해줘.

# How to Use


1. 이미지를 캡쳐하세요.
2. 채울 노드를 선택하세요.
3. 이 프롬프트를 복사하고 MCP 인터페이스에 붙여넣기 해서 실행하세요.
4. 실행 중..
5. 결과 확인
