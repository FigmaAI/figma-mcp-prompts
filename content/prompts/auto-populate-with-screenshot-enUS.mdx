# Prompt

Scan all texts from the selected nodes, extract the site map text from the attached image, and update all texts at once.

# How to Use

1. Screen capture image 
2. Select nodes to fill 
3. Run the prompt: Copy and paste this prompt into the MCP interface 
4. Running 
5. Result
