# Prompt

- 피그마에 접속하여 선택 노드를 가져옵니다.
- 레거시 주석을 변환하고 피그마 주석으로 교체하자.
- 우리는 사전에 설정된 변환 전략이 있으니깐 이것에 따라서 작업해주면 됨.

# How to Use

> 🚨 Attention please
> 주석 기능은 유료 플랜인 프로 이상에서만 제공됩니다.
> ![Attention](/assets/attention-please.png)



1. 변환할 기획서 페이지를 선택하세요
2. 이 프롬프트를 복사하고 MCP 인터페이스에 붙여넣기 해서 실행하세요.
3. 실행 중...
4. 결과 확인
