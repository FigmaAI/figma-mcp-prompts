# Prompt

- Find footer site-map from @https://www.grab.com/sg/
- Extract all text from there
- scan all texts from the selected nodes 
- and update all texts at once.

# How to Use

1. Copy url
2. Select nodes to fill
3. Run the prompt: Copy and paste this prompt into the MCP interface
4. Running
5. Result
