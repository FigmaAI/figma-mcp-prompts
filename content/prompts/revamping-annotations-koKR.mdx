# Prompt

- 피그마에 접속해서 선택 사항을 가져옵니다.
- 선택된 노드의 하위 노드 구조를 파악합니다.
- 선택된 노드 내부의 기존 주석만 가져옵니다.
- 각 주석을 적절하게 분류합니다.
- 주석 내용을 Markdown 형식으로 개편하고 요약을 추가합니다.


# How to Use

> 🚨 Attention please
> 주석 기능은 유료 플랜인 프로 이상에서만 제공됩니다.
> ![Attention](/assets/attention-please.png)


1. 주석을 추가할 노드를 선택하세요
2. Run the prompt: Copy and paste this prompt into the MCP interface
3. Running
4. Result
